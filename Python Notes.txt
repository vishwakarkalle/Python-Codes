#Deep copy & shallow copy methods in Python:-
===========================================
1) Shallow copy methods:-
In the shallow copy method it can copy/add the any elements to its oringal list but the values which are present inside the list are not changed or affected it remains the same, it only adds the inside the list values.
original=[53,24,12,87[91,09,18],100,03]
when we apply the shallow copy it applied only to values which are inside the list not inside the nested values.
and to add the values into to nested list inside the list we need to use the "DEEP COPY".

2) Deep Copy Method:-
Here we use the deep copy in the nested Mutable list/objects to add any element then this deep copy method add the given values to the deep copy list not to the original list, here the original copy lists remains the same.

When to use Shallow Copy and Deep copy:-
======================================
 Scenerio                                            Use
 1) Flat(simple) list/ dictionary(no nesting)      1) Use shallow copy(faster)
 2) Nested data u wil modify                       2) Deep copy(safe)/(slower)
 3) Only modifying the outer level                 3) Shallow copy
 4) modifying ANY nested level                     4) Deep copy
 5) Performance critical + no nesting              5) Shallow Copy

 Will i touch anything nested ? "Yes" --> Deep copy
                               "NO"  --> Shallow Copy

Inheritance:-
-----------
Inheritance lets a new class(the child) automatically re-use the atrributes and methods of an existing class(the parent).

Real world analogy:-
-----------------
1) A car is a Vehicle:- every car shares an vehicle properties(wheels,engines,fuel)
2) A Dog(is-A animal):- every dog inherits with animal traits.

why use inheritance:-
====================
1) Code Re-use:- write once, inherit everywhere.
2) Modeling real relationships:- capture "is-a" hirearchy naturally.
3) Easier maintaince:- fix a bug in parent and all childrens get benifitted.

TIP:- Use inheritance when there's true "is-a" relationship. If its just 'has-a', use composition instead.

To inherit, put the parent class in paranthesis after the child class name.
class Parent:
    pass                    #This is basic syntax of the inheritance.
class Child(Parent):
    pass
