Functions in Python:-
===================
It's a reusable block of code that performs the specific task.
Instead of writing the code same again and again, you write it once inside a function and call it whenver you need it.

This follows the "DRY" principle - "Don't repeat yourself" its one of the important principles in the python programming.
This Functions makes our code easier to read,easier to debug and easier to maintain.
If the logic needs to change, you can change it in one place not everywhere.

How to define & call the fuction:-
---------------------------------
You define a function using the "def" keyword followed by the function name and parentheses.
You call it by writing its name followed by parentheses.

#defining a function
def function_name():
     # code goes here

#calling a function:
function_name()

Best example without using the function:-

Return statement:-
================
Here the "Return" statement sends a value back to whoever called the function. Without "return" a functionjust does something
but gives nothing back. With return, you can use the result in therest of your program.

"Note":- Once Python hits a return statement, the function stops immediately.
Any code after return inside the function will never run.
in simple words, anything which is written after the "return function" , it wont excute at all & it will simply ignore everything and prints the output.

#Slicing in Python:-
================
Slicing is a powerful feature in python that allows you to extract a specific portion or slice from a list.
it doesn't modify the original list;instead, it returns a new list containing only the selected element.
This is incredibly useful for working with subsets of data effeciently.

Slicing Syntax:-
===============
The General index for slicing is 'list_name[start:stop:step]':
1) start index  2) stop index     3) step value

list methods in python:-
======================
1) Append :- means adding the values in the list at the end. ex:- lst.append(100)
2) Insert:- means adding the string or any values in the list by its index value  ex:- lst.insert('Apple')
3) Remove:- means removing the values from the list.  ex:- lst.remove(23)
4) Pop():-  means it removes an element from a specific data collection(like a list or a dictorinary) and returns that removed value.
5) Sorting:- It gives the data elements output in the ascendig order. nums(sort) means ascending order and nums.sort(reverse=True) means - It gives the data elements output in the descending order.
6) Reversing:- It reverses the order of elements in the list directly, modifying the original list.   ex:- num.reverse()
7) Searching:- It returns the index value of the specific item/element from the list.
8) Counting:- It counts the how many times the any value or element is repeated in the list and it gives the output.

Membership Operators:-
=====================
Membership Operators are used to test if a sequence(like a list) contains a specified element. They return a boolean value(True or false) based on the result of the check, making them highly useful for conditional logic.
i) in - Check for Membership:- This operator returns if the specified value is found within the list, and false otherwise.
ii) not in - Check for Non-Membership:- This operator returs True if the specified value is NOT found within the list, and False otherwise.

S7) Tuples:-
===========
Tuples is a collection of values stored in a single variable, very similar to the list.
The key difference is that a tuple is immutable(cann't modify elements once its created or change,add or remove its elements).
its similar to Lists. But in list we use [] square bracket where as here we use () paranthesis.

Key properties of Tuple:-
==========================
1) List can be changed, but tuples Can't be changed once its created.
2) Ordered- items stay in the order you put them in tuple.
3) Immutable- Can't be modified/created/changed.
4) Allows Duplicates- The same value can be appear more than once.
5) Can hold any data type:- integers,string,boolean values etc
